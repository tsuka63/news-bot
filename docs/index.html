🧭 マクロ → セクター影響リファレンス

利上げ・為替などが「どのセクターに有利／不利か」の教科書的な傾向。
マクロ要因追い風 ◎逆風 ▼補足
利上げ（金利↑）銀行・保険（利ざや拡大）グロース／ハイテク・REIT・公益（割引率↑で割高に）通貨高（円なら円高）要因。ただし想定内なら織り込み済みで動かないことも。
利下げ（金利↓）グロース／ハイテク・REIT・不動産銀行（利ざや縮小）通貨安（円安）要因。景気刺激で全体的にはプラスに働きやすい。
円安（ドル円↑）輸出株（自動車・電機・機械・半導体製造装置）輸入・内需（小売・電力・食品）海外売上比率が高い企業ほど追い風。
円高（ドル円↓）輸入・内需輸出株外貨建て資産（米国株）は円換算で目減り。
原油・資源高エネルギー・商社・資源（石油・非鉄）運輸・素材・電力（コスト増）インフレ加速→利上げ観測につながることも。
